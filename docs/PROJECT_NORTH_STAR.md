# PROJECT_NORTH_STAR
更新时间：2026-04-15

## 最终目标

**一句话：** 把 easyuse.ai 打造成"电商卖家做图的第一选择"

**输入：** 卖家上传产品图 / 答完诊断题

**输出：** 高质量电商产品图（白底主图 / 模特图 / 场景图）+ 留资转化

---

## 核心赚钱路径

```
流量进入 → 首页吸引 → 诊断问卷 → 结果页展示价值 → 上传产品图 → 留资/付费
```

---

## 产品定位

- **不做：** AI文案工具、内容引流、客服自动化
- **只做：** AI电商作图服务（服装类目为主）
- **核心卖点：** 不用学AI / 不用自己折腾 / 上传就能出图

---

## 核心功能

1. **首页** — 展示before/after效果，3秒内明白"这是帮我做图的"
2. **诊断问卷** — 3题快速判断用户类型，给出个性化方案
3. **结果页** — 根据诊断类型展示对应内容和服务流程
4. **留资/咨询** — 最低摩擦留微信，48h内顾问联系
