# PROJECT_NORTH_STAR
更新时间：2026-04-14

## 最终目标

**输入：** 用户答完5道题

**输出：**
- 用户画像（结构化）
- 匹配工作流（12选1）
- 图案设计方案（结构化）
- 内容文案
- CTA

## 系统链路

```
用户答题 → 五道题 → 用户画像 → 工作流匹配 → 图案/内容生成
```

## 核心赚钱路径（来源：ops/SYSTEM_GOAL.md）

```
流量进入 → 上传商品图 → 生成模特图 → 预览价值 → 触发转化 → 留资/付费
```

| 步骤 | 当前状态 | 优化方向 |
|------|----------|----------|
| 1. 用户进入 | ✅ 首页可访问 | 降低跳出率 |
| 2. 上传商品图 | ✅ 可上传 | 引导文案更清晰 |
| 3. 生成模特图 | ✅ API 200 + 真实OSS图片 | 成功率 95%+、出图速度 |
| 4. 结果页体验 | ✅ 可展示 | 图片更大、更清晰、加载更快 |
| 5. 触发转化 | ❓ CTA 存在但未测试 | 文案更吸引、位置更优 |
| 6. 留资/付费 | ❌ 未测试 | 路径最短化 |

**当前最大瓶颈：步骤 5（CTA 转化）**

## 核心模块

1. **问卷系统** - 五道题表单 + 提交
2. **画像生成** - LLM 根据答案生成用户画像
3. **工作流匹配** - 答案 → 12工作流映射
4. **图案生成** - 工作流 → Prompt → 图片
5. **内容生成** - 工作流 → 文案/CTA

## 技术栈

- Next.js 15 + TypeScript
- MiniMax API（对话 + 图片）
- Playwright（自动化测试）
- lib/diagnosis-workflow-map.ts（映射层）
