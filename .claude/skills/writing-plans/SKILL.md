# writing-plans

> 结构化写作计划模板：提供文章、文档、方案的标准化写作流程和模板，提升写作效率和质量。

## 文章写作流程

### Step 1 — 明确目标

```
写作前必答：
1. 这篇文章的目标是什么？（教育/说服/娱乐/通知）
2. 目标读者是谁？（技术/管理层/大众/特定人群）
3. 文章需要多长？（短文/中篇/长文/系列）
4. 最终交付格式？（Markdown/Word/PPT/邮件）
5. 截止时间是什么时候？
```

### Step 2 — 收集素材

```
素材类型：
- 内部数据/案例
- 外部研究/报告
- 个人经验/观点
- 用户反馈
- 竞品分析

收集渠道：
- 数据库/报表
- 竞品官网/文档
- 行业报告
- 访谈记录
```

### Step 3 — 确定结构

```
文章结构选择：

1. 总分总结构（通用）
   - 开篇：引入话题
   - 分述：3-5 个要点
   - 总结：核心结论

2. 解决问题结构（教程/指南）
   - 问题背景
   - 解决方案
   - 实施步骤
   - 注意事项

3. 时间线结构（历程/回顾）
   - 过去：背景和挑战
   - 现在：做了什么
   - 未来：计划和展望

4. 对比结构（评测/选型）
   - A 方案介绍
   - B 方案介绍
   - 对比分析
   - 建议选择

5. 金字塔结构（商业报告）
   - 核心结论（最重要）
   - 关键论点（3 个支撑）
   - 详细证据（数据/案例）
```

### Step 4 — 撰写大纲

```
大纲模板：

# [主题]

## 一、[第一要点]（15%）
### 1.1 [子要点]
### 1.2 [子要点]

## 二、[第二要点]（40%，重点展开）
### 2.1 [子要点]
### 2.2 [子要点]
### 2.3 [子要点]

## 三、[第三要点]（30%）
### 3.1 [子要点]
### 3.2 [子要点]

## 四、[总结]（15%）
### 核心结论
### 行动建议
### 后续展望
```

### Step 5 — 填充内容

```
写作原则：
- 每段只说一件事
- 观点先行，论据跟上
- 使用具体数字和案例
- 避免模糊表达
- 段落长度 3-5 句为宜

连接词使用：
- 表顺承：此外、而且、再者
- 表转折：然而、但是、不过
- 表因果：因此、所以、于是
- 表举例：比如、例如、以 xxx 为例
```

### Step 6 — 审核修改

```
自检清单：
□ 标题是否清晰、有吸引力？
□ 开头是否钩住读者？
□ 每个观点是否有证据支撑？
□ 逻辑是否连贯？
□ 是否有错别字或病句？
□ 格式是否统一？
□ 配图是否合适？

修改优先级：
1. 逻辑和结构（大纲是否合理）
2. 内容准确性（数据/事实是否正确）
3. 表达清晰度（是否易于理解）
4. 文字流畅度（是否通顺）
5. 格式规范（排版/配图）
```

---

## 限制条件

- 模板是起点不是终点，根据实际需求调整
- 技术文档要准确，避免误导读者
- 商业文档要考虑受众和商业影响
- 保持更新，文档有保质期
- 尊重版权，引用要注明来源
